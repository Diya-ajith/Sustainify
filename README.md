# 🌿 Sustainify: Proactive Community Sustainability Companion

Sustainify is a **web-based sustainability companion powered by agentic AI** that transforms passive sustainability platforms into **proactive, personalized assistants**.

Aligned with **SDG 11: Sustainable Cities and Communities**, Sustainify empowers users to discover local sustainability events, participate in eco-friendly activities, and build sustainable habits with minimal effort.

---

# 🎯 Sustainable Development Goal

### SDG 11: Sustainable Cities and Communities
Make cities and human settlements **inclusive, safe, resilient, and sustainable**.

Sustainify contributes to this goal by encouraging **community participation in environmental initiatives** and helping people easily discover sustainability opportunities around them.

---

# 💡 Core Features

### 🧠 Proactive Opportunity Matching
Automatically scans local databases and identifies **relevant sustainability events and activities** based on user interests.

### 📅 Automated Scheduling
Allows **one-click event registration** and automatically adds events to the user’s **calendar for reminders and planning**.

### 📬 Weekly Personalized Digests
Sends **personalized email digests** containing:
- Upcoming eco-events
- Sustainability tips
- Environmental news

# ⚙️ How It Works

1. Users submit their **interests and location preferences** through the website form.
2. The **AI agent processes the user data** and matches it with sustainability opportunities.
3. The system periodically scans **event sources and local databases**.
4. The **scheduler triggers notifications and weekly digests**.
5. Users receive **email alerts about relevant sustainability activities**.
